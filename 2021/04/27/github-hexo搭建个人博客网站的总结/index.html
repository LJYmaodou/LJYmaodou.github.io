root: /

上传图片

方法1

参考教程
注意站点目录指得是根目录，而主题目录指的是Next主题所在目录，如 D:\Blog\themes\next。而这两个不同的目录都有一个配置文件_config.yml。
在主题目录中找到 source/images目录（绝对路径 D:\Blog\themes\next\source\images）将所需要上传的图片复制到该目录下即可。在.MD文件中直接引用即可。

方法2

对于大量图片且希望和.md文件整理在一起的话，可以采用第二种方法上传图片。
将_config.yml文件中的配置项post_asset_folder设为true后，执行命令$ hexo new post_name，在source/_posts中会生成文章post_name.md和同名文件夹post_name。将图片资源放在post_name中，文章就可以使用相对路径引用图片资源了。
